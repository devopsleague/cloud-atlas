交互执行
===========

``irb`` 提供了交互方式运行 Ruby 脚本命令，类似于 ``python`` 命令不带参数进入交互模式::

   irb

测试上述简单执行命令::

   irb(main):001:0> print("Hello, Ruby.\n")
   Hello, Ruby.
   => nil
   irb(main):002:0>

``nil`` 是 ``print`` 方法的返回值

.. note::

    Ruby在调用方法时候可以省略 ``()`` ，所以::

       print("Hello, Ruby.\n")

    和::

       print "Hello, ruby.\n"

    是一样的(类似 :ref:`shell` )

单双引号区别
===============

和 :ref:`shell` 类似， ``''`` 会忽略字符串中 \\ 转义符号， ``""`` 则能够处理类似 ``\n`` 这样的特殊转义符号
