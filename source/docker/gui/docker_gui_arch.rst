==================
Docker GUI架构
==================

通常我们都采用Docker来运行服务器应用，不需要在容器内部安装任何GUI相关组件，例如X11。

不过，如果要快速提供标准化图形工作环境，一键部署图形应用，通过客户端访问运行容器中的GUI应用，Docker容器化是一个非常巧妙的轻量级解决方案，适合Linux GUI应用程序。
