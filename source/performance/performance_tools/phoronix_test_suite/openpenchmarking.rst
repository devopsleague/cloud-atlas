====================
OpenBenchmarking
====================

`OpenBenchmarking.org <https://openbenchmarking.org>`_ 是 ``Phoronix Test Suite`` 配套的WEB服务，提供用户将自己硬件和软件benchmarks通过在线接口发布以及共享的网站服务。在这个网站上，可以将不同软件和硬件的性能进行横向对比，方便进行性能评估以及持续改进。
