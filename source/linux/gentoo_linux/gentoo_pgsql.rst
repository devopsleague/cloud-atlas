=====================
Gentoo PostgreSQL
=====================

:ref:`pgsql` 是著名的开源关系型数据库(relational database management system, RDBMS)，支持事务(transactions), 模式(schema)和外键(foreign keys)等功能，并且通常被认为比其他数据库(无论是商业数据库还是开源数据库)在默认情况下更为严格地遵守SQL标准，也更为安全。
