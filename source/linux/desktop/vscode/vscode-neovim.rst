=====================
vscode-neovim
=====================

``VSCode Neovim`` ( `GitHub: vscode-neovim/vscode-neovim <https://github.com/vscode-neovim/vscode-neovim>`_ )提供了VSCode中原生使用的 :ref:`nvim` ，为VSCode带来了全新、全面的 :ref:`vim` 体验。

结合两种优秀的编辑器/IDE，特别适合轻量级键盘操作桌面( :ref:`sway` / :ref:`i3` )
