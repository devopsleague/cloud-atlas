==================
vim-snipmate
==================

`SnipMate <https://github.com/garbas/vim-snipmate>`_ 提供了自动在代码中插入片段方便开发。在 :ref:`vimrc` 中已经包含这个插件，可以非常方便使用:

- 在C语言程序编写时，输入 ``for`` 然后按下 ``<tab>`` 会自动扩展成一段for循环代码
