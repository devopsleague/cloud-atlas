=================================
Linux平板
=================================

Windows平板基本都是x86体系，可以完美兼容PC桌面应用。我偶然看到淘宝上有很多非常廉价的Win10平板，由于是x86体系(atom处理器)，实际上也支持安装Linux，网上有一些资料可以参考。很多二手Windows平板价格低廉，如果可以安装Linux，我准备hack了安装双系统(或者纯Linux)，定制日常看书，并且外接键盘做一些移动开发工作。
