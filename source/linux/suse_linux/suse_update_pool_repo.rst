========================
SUSE本地更新和软件池仓库
========================

SUSE注册用户可以从SUSE客户中心订购(下载同步)更新软件仓库，在局域网内部构建一个完整的补丁和软件仓库，提供局域网内部所有SUSE服务器使用。
