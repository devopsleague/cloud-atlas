=====================
Cloudflare基础架构
=====================

随着我准备实践Cloud Atlas，考虑到需要CDN以及Tunnel，准备研究和私用Cloudflare免费服务:

- Cloudflare Tunnel
- Cloudflare Pages
- Cloudflare CDN

Cloudflare最大的优势是遍布全球的CDN加速，对于非关键型个人或业余爱好项目是免费的(一个网站)，参考 `Cloudflare计划与价格 <https://www.cloudflare.com/zh-cn/plans/>`_
