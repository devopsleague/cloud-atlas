TypeScript起步
========================

TypeScript和JavaScript关系
============================

- TypeScript源码编译输出的是JavaScript源码，在编译阶段的AST(抽象句法树, abstract syntax tree, AST)会进行类型检查:

  - TypeScript源码 -> typeScript AST
  - 类型检查器检查AST
  - TypeScript AST -> JavaScript源码
