=========================
Intel酷睿CPU散热器
=========================

Intel处理器官方配置了原装散热器:

- ``LGA115x`` TDP<=65W

- ``LGA115x`` TDP<=95W: 在纯铝散热器中心增加了一个铜芯(其实只有一个薄块)来提升散热效果(不过高功耗版本i5/i7也比较吃力)

- ``LGA1366`` TDP<=95W: 即X58旗舰平台，通过 "加厚、加大、加铜芯" 版本元转散热器 (高度应该超过了5cm)
