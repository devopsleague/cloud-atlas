=======================================
在反向代理后面运行Grafana (sub-path)
=======================================

在配合 :ref:`prometheus_startup_centos7` ，也采用了相似的 ``sub-path`` 反向代理：

- 在 :ref:`install_grafana` 完成后，对于反向代理，需要修订 ``/etc/grafana/grafana.ini`` 配置文件:

  - 访问域名 ``domain``
  - 访问url ``root_url``
