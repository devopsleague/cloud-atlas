========================
Cilium Cluster Mesh
========================

Cilium Cluster Mesh提供了将多个Kubernetes集群连接到一起的网格化能力，实现跨所有集群的 ``pod-to-pod`` 连接，可以在不同集群间定义全局服务和负载均衡，以及增强型的访问安全策略。
