============================
ARM移动云计算的DNS和NTP服务
============================

对于分布式集群系统，即使在一台单一物理主机上部署虚拟化集群，也需要解决模拟集群的DNS解析和NTP同步，否则部署过程会遇到很多困难:

- :ref:`kvm` 虚拟化非常容易出现时钟偏移，模拟主机的时间即使只有秒级差异也会导致分布式系统运行异常
- 随着虚拟主机的增加，各个主机之间的域名解析同步是无法通过手工完成的，非常有必要构建一个集中的层级DNS服务器
