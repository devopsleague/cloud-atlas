=========================
LLaMA在CPU架构上优化
=========================

:ref:`install_llama_cpu` 支持基于CPU的优化，探索这项技术主要目标:

- 利用廉价的(二手淘汰服务器)CPU和大内存来实现基本能够运行的大模型
- 学习如何优化大模型运行的基础设施部署
