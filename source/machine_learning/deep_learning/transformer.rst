=========================
Transformer模型
=========================

Transformer模型(直译为"变换器")是一种采用自注意力机制的深度学习模型，这个机制可以按照输入数据各部分重要性的不同而分配不同的权重。Transformer模型主要用于 "自然语言处理" (NLP) 与 "计算机视觉" (CV) 领域。
